# Zhip
Zhip is the worlds first and only iOS wallet for Zilliqa. The app is entirely open source. It uses the [Zilliqa Swift SDK "Zesame"](https://github.com/OpenZesame/Zesame).

## Installation

### Prerequisites
1.  macOS High Sierra or [Mojave](https://itunes.apple.com/se/app/macos-mojave/id1398502828?l=en&mt=12)
2.  [Xcode 10](https://itunes.apple.com/se/app/xcode/id497799835?l=en&mt=12)

### Get started
1.  `git clone` this repo, `cd` to the newly cloned folder 
2.  `open Zhip.xcworkspace`
